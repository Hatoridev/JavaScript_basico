Como Funciona o Loop For?

O loop for é uma estrutura de repetição que executa um bloco de código várias vezes. Ele é geralmente utilizado quando sabemos o número de vezes que queremos repetir um processo. O loop possui três partes principais:

  • Inicialização: Define a variável de controle.
  • Condição: Define a condição para continuar a execução.
  • Incremento: Define como a variável de controle será alterada após cada iteração.
